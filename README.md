## Image Resizer Tool

## 1. Objective:

To resize and convert multiple images automatically using Python.

## 2. Tools Used:

## Python

 Pillow (PIL) library

## 3. Description:

This project resizes a batch of images in one go.

The program reads all images from a specific folder.

It resizes them to a fixed dimension.

The new images are saved in a separate output folder.

It helps automate image resizing tasks efficiently.

## 4. Working Steps:

Place all images inside a folder named images.

Run the Python script (app.py).

The program will automatically create another folder named resized_images.

All resized images will be saved in the resized_images folder.

## 5. Hints:

Use the os module to read all files from the folder.

Use the PIL.Image module to resize and save the images.

## 6. Outcome:

Automates image resizing and conversion.

Saves time and ensures all images are of uniform size.
